# Repository Instructions

## Media Policy

- Never generate, request, add, or publish AI-generated images.
- Do not use image-generation tools for this project.
- Use only media supplied by the portfolio owner or media with a documented
  license and source.
- When real media is unavailable, render a clearly labeled interface
  placeholder using HTML, CSS, and the existing icon library.
- Do not use remote stock-photo placeholders as substitutes for owner content.
- Remove image metadata and precise location data before publishing personal
  photographs unless the owner explicitly chooses to retain it.
